Projects

  • CISC 488 | Clinical Named Entity Recognition and Similarity Analysis for Predicting Patient Diagnoses
    Clinical notes are rich in unstructured data, containing information about a patient’s condition, symptoms, and treatment. This project aims to extract medical entities from clinical text using NER automatically, and to evaluate the semantic similarity between clinical case descriptions to identify related diagnoses.

  • CISC 483 | Clinical Decision Support via Integrated Patient Behavior and Risk Pattern Mining.ipynb
    This project aims to build a Clinical Decision Support System that integrates multiple data mining techniques to extract interpretable patterns and recommendations from electronic health records (EHRs), focusing on oncology patients with comorbid conditions. The goal is to identify both latent clusters of patient behavior and explicit associations between symptoms, diagnoses, treatments, and outcomes. By unifying clustering, graph-based ranking, association rule mining, collaborative filtering, and decision tree classification, this project will offer a modular, data-driven framework for real-time clinical decision support.
